# Contributing

Thanks for helping improve MakePay by MakeCrypto.

- Public issues and pull requests are welcome.
- Keep pull requests focused and include tests or documentation when behavior changes.
- Do not include API keys, package-manager tokens, customer data, or other secrets.
- MakePay by MakeCrypto maintainers review all changes, decide what gets merged, and choose release timing.
- Releases and version publishing are maintainer-only; an accepted pull request does not imply an immediate package release.
