# Security Policy

Please report suspected vulnerabilities privately.

- Use GitHub private vulnerability reporting from this repository's Security tab when available.
- If private reporting is unavailable, email support@makecrypto.io with the subject `Security: MakePay`.
- Include affected versions, impact, reproduction steps, and any proof of concept that helps us verify the issue.
- Do not open public issues or pull requests that disclose exploitable security details.

MakePay by MakeCrypto maintainers will triage reports, coordinate fixes, and publish security guidance when appropriate.
